Document

Document object represent the web page in the browser window. It provide access to HTML content and allowing to manipulate and interact with the document's element. we can access and modify the document's elements, properties and methods. Common operations that can be performed such as selecting elements with JS , changing their content and responding to user events.

window object

Window object is the top level object in the browser's JS object hierarchy. It represents the browser window or tap containing web page. We can access the window object directly by using `this` keyword within the global context. window object provides properties and methods for controlling and interacting the browser window such as , opening and closing windows, navigating to URLs etc.. it also containing global variables and functions such as ` setTimeout` , `setIntervel` and the `console` object.
